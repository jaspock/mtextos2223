GPT significa ‚ÄúGenerative Pretrained Transformer‚Äù. Es un modelo de lenguaje que utiliza t√©cnicas de deep learning para generar texto de manera aut√≥noma. GPT ha sido entrenado en una amplia cantidad de contenido textual.

A continuaci√≥n se muestra un ejemplo de uso de GPT2:

from transformers import GPT2Tokenizer, GPT2LMHeadModel

# Inicializa el tokenizador

# Inicializa el modelo
model = GPT2LMHeadModel.from_pretrained('gpt2')

# Define el prompt
prompt = "Escribe un ejemplo sobre la importancia de la inteligencia artificial"

# Tokeniza el prompt
input_ids = tokenizer.encode(prompt, return_tensors='pt')

# Marca el fin de la entrada
input_ids = input_ids[:, -1].unsqueeze(0)

# Activamos el modelo en modo evaluaci√≥n
model.eval()

# Generamos el texto
with torch.no_grad():
    outputs = model(input_ids, labels=input_ids)
    loss, logits = outputs[:2]
    logits = logits[0, -1, :]
    probs = torch.nn.functional.softmax(logits, dim=-1)
    next_token_id = torch.argmax(probs).unsqueeze(0)
    input_ids = torch.cat([input_ids, next_token_id], dim=-1)

# Convertimos los ids a texto
generated_text = tokenizer.decode(input_ids[0].tolist())

# Imprimimos el resultado
print(generated_text)
